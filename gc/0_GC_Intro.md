# jvm-memory

## 1. memory 란?

컴퓨터 에서 데이터 를 저장 하는 장치 이다.  
memory 는 데이터 를 저장 하는 장치 로서, 데이터 를 저장 하고 읽어 오는 기능을 수행 한다.  
memory 는 disc 보다 데이터 를 빠르게 읽고 쓸 수 있으며, 데이터 를 저장 하는 용량 이 작다.
